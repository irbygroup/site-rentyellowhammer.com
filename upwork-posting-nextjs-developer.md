# Next.js Developer — Build Event Venue Website from Figma Design (1 of 3 Venues)

We're Yellowhammer Hospitality, a boutique event venue company in Mobile, Alabama with 3 properties. We have a **complete Figma design** ready and need a Next.js developer to implement it — starting with **The Courtyard on Dauphin** (courtyardondauphin.com) as Phase 1.

## The Big Picture

We have 4 domains:
- **rentyellowhammer.com** — Corporate hub linking to all 3 venues
- **courtyardondauphin.com** — The Courtyard on Dauphin (Phase 1)
- **oakandfountain.com** — Oak & Fountain (Phase 2)
- **hallettirby.com** — The Hallett-Irby House (Phase 2)

All 3 venue sites share the same Figma template/design. Phase 1 is implementing the Courtyard site to 100% completion. Phase 2 (separate contract, same developer preferred) will replicate it for the other 2 venues and build the corporate hub page.

## What You'll Receive

- **Complete Figma design** — all pages at desktop, tablet, and mobile breakpoints with a component library and style guide
- **Photo assets** — extensive professional venue photography
- **All copy/content** — page text, testimonials, venue details
- **Chatwoot account** credentials
- **Vercel account** access
- **Domain** access

## Phase 1 Scope — courtyardondauphin.com

Implement the Figma design as a Next.js site with these pages:

1. **Home** — Hero video/imagery, "Book a Tour" CTA, phone number, venue highlights, reviews
2. **The Venue** — About, location, history, photo highlights
3. **Weddings** — Ad landing page targeting wedding searches
4. **Parties & Personal Events** — Ad landing page for baby showers, birthdays, anniversaries, reunions
5. **Corporate Events** — Ad landing page for corporate retreats, meetings, team building
6. **Photo Gallery** — Image grid with lightbox
7. **Pricing** — Package overview with lead capture form
8. **Contact** — Lead capture form, phone, email, Google Maps embed

## Technical Requirements

- **Next.js 15** (App Router) + **Tailwind CSS** + **TypeScript**
- **Deployed on Vercel** with automatic preview deployments
- **"Book a Tour" CTA** prominent on every page — header, floating mobile button, inline CTAs
- **Click-to-call** phone number (251-333-RENT) prominent throughout
- **Lead capture forms** on Pricing and Contact pages (name, email, phone, event type, date, message) — submissions sent via email (SendGrid or similar) and pushed to our CRM (FollowUpBoss API)
- **Chatwoot** live chat widget integrated
- **Photo gallery** with lightbox/modal viewer
- **SEO:**
  - Semantic HTML with proper heading hierarchy
  - Unique meta titles and descriptions per page
  - Open Graph and Twitter Card tags
  - JSON-LD structured data (LocalBusiness + EventVenue schema)
  - Sitemap.xml and robots.txt
  - Clean URL structure with trailing slashes
- **Performance:** Target 90+ Lighthouse scores (Performance, Accessibility, Best Practices, SEO)
- **Mobile-first responsive** — pixel-perfect match to Figma at all 3 breakpoints
- **Cloudflare Turnstile** on forms (spam protection)
- **Google Analytics / Tag Manager** integration
- **Reviews/testimonials** displayed across all pages
